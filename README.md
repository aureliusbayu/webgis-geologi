# Project Folder Structure

Please organize your project files as follows:

```
/ (Root Directory)
├── css/
│   └── style.css          # Global custom styles (fonts, colors, map overrides)
├── js/
│   └── map-logic.js       # Leaflet and PapaParse logic
├── assets/
│   ├── images/
│   │   ├── logo.png       # The GeoAccess logo
│   │   └── bamboo.png     # The bamboo illustration
│   └── icons/             # Social media icons (Instagram, LinkedIn)
├── index.html             # Landing Page
└── map.html               # Web Map Page
```

**Note:** The `style.css` file provided handles the custom colors and font integration, designed to work alongside Tailwind CSS (CDN).
